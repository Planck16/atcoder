・掘削方針
  家から水源0まで
  家からマンハッタン距離が最小となる水源まで -> スコアが大きく上がった
  家から最短の水源・水路まで
  家から水源までの距離が小さいものから掘削 -> スコアが大きく上がった
  掘削を終えるごとに、家から水源・水路までの距離を求め、小さいものから掘削 -> スコアが少し上がった
  岩盤の硬さを記憶しておき、掘削経路に固い岩盤がありそうなら別の掘削経路を探す -> スコアが少し下がった　意味ない
  掘削を終えるごとに、家から水源と「他の家」までの距離を求め、最短の場所まで掘削 -> 意味ない
  家から水源までの距離が近い座標を3つ記憶、その中からcostが最小となるような水源まで掘削

  ランダムな地点を掘削し、岩盤の硬さの分布を得る
  距離と岩盤の硬さを考慮に入れた値で家から掘削する場所を決める
    dist := 100(dx+dy) + (sx+xy)
    sx := x軸に平行な経路の中間にある岩盤の硬さ

・power
  power = 100
  if (c <= 8) power = 50;
  else if (c <= 64) power = 100;
  else power = 200;


・cの値
  cが小さいときpowerを小さく、大きいときpowerを大きくしたらスコアが上がった


・岩盤の硬さ
  岩盤の硬さに応じてpowerを変える
    固いときpowerを大きく、やわらかいとき小さく
  二分探索で最適なpowerを求める
  岩盤の硬さを記憶する
    前回掘削した岩盤の硬さを記憶し、それをもとにpowerを定める
